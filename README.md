# Kotlin Playground [![Build Status](https://travis-ci.org/icoffiel/kotlin-playground.svg?branch=master)](https://travis-ci.org/icoffiel/kotlin-playground) [![Coverage Status](https://coveralls.io/repos/github/icoffiel/kotlin-playground/badge.svg?branch=master)](https://coveralls.io/github/icoffiel/kotlin-playground?branch=master)
A repository to play around with Kotlin and Spring
## Running
The application can be started using the following:
```
gradlew bootRun
```

The app will then be available at `localhost:8080`
## Testing
Tests can be run using the following:
```
gradlew test
```
